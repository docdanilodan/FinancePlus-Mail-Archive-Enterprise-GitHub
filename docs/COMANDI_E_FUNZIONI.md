# Comandi e funzioni

| Pulsante/Pagina | Funzione | Output |
|---|---|---|
| Dashboard | Riepilogo mail, clienti, allegati, temporanei | KPI e tabella |
| Scarica mail > Cerca mail | Cerca su Gmail per date, mittenti, allegati | Anteprima mail |
| Scarica e salva selezionate | Salva corpo email e allegati | Cartelle cliente/anno/mese |
| Scarica tutte | Scarica tutto il blocco trovato | Archivio completo blocco |
| Sposta nel cestino dopo salvataggio | Elimina da Gmail solo dopo salvataggio DB | Mail in cestino Gmail |
| Archivia dopo salvataggio | Rimuove da Inbox solo dopo salvataggio DB | Mail archiviata |
| Mail salvate | Mostra DB delle mail archiviate | Tabella operativa |
| Clienti | Mostra anagrafica creata automaticamente | Elenco clienti |
| Documenti da abbinare | Mostra file non riconosciuti | `_TEMP_DA_ABBINARE` |
| Report | Esporta CSV e ZIP archivio | Download file |
